Welcome to Tauc-Urbach Extractor v1.0

This software is specifically designed for extracting the bandgap using Tauc methods and Urbach energy using Urbach methods. It analyzes transmittance data obtained from UV-visible spectroscopy measurements.

Developed by:
Moch. Januar, Ph.D.,
LIU Group, Department of Electronic Engineering
Chang Gung University

I. Step-by-Step User Guide:

  1. Starting the Program: Launch the application by double-clicking "TaucUrbach_Extractor.exe."
  2. Data Preparation: Prepare an ASCII file (*.txt or *.csv) containing two columns:
    1. wavelength in nanometers [nm] and
    2. transmittance data in percentage [%].
  3. Loading Data: Drag and drop the file or click the 'Open File' button to import the transmittance data into the program.
  4. Initial Data Check: View the raw data and ensure it is displayed correctly.
  5. Data Configuration: Enter the thickness of the film in nm. Adjust the 'skip-point' setting (used only for plotting) to enhance the visual presentation of the experimental data.
  6. Update Figures: Click 'Update Figure' button to refresh the figure after performing some settings.
  7. Data Fitting: Click the 'Fitting' button to start the data fitting process.
  8. Adjusting Fit Range: Fine-tune the fitting range for both Tauc and Urbach methods by selecting the most linear region of the experimental data. This adjustment is made using the dial buttons (for fast tuning) or the spin box (for fine tuning) located below each respective panel.
  9. Reviewing Results: Check the line-edit panels at the bottom for the values and standard deviations of the calculated bandgap and Urbach energy.
  10. Update Figures: Click 'Update Figure' button to refresh the figure after the data fitting process.
  11. Fine-Tuning: If necessary, further refine the fitting range to achieve a lower standard deviation in the results for the bandgap and Urbach energy by repeating the above fitting procedures.
